💡 Problem Statement: Double Operation Number Series Analyzer
📌 Title
Detection and Prediction of Double Operation Number Series

📖 Description
A number series is given where elements follow a hidden pattern. In some series, two operations repeat alternately (for example: multiplication and subtraction).
The task is to design and implement a program that:
Identifies the alternating operations in the series
Validates whether the pattern is consistent
Predicts the next number in the sequence

🎯 Objective
Develop a Java program to:
Analyze a given integer array
Detect a double operation pattern (like ×n and −m alternately)
Verify correctness of the pattern
Generate the next number in the series.
